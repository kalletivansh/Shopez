# ShopEZ — MERN Stack E-commerce Application

ShopEZ is a full-stack MERN E-Commerce application featuring JWT-based authentication, role-based access control, product CRUD operations, shopping cart management, order tracking, and an admin dashboard. Built with React.js, Node.js, Express.js, and MongoDB, it provides a secure and responsive shopping experience.
